Cómo compilar y ejecutar:
1. Abrir el archivo Makefile
1. Ejecutar el comando make dependiendo del tipo de paralelización: 
* `make serial`
* `make omp`
* `make mpi`
* `make vectorization`.
3. El archivo ejecutable se encontrará en la carpeta `build/<estrategia>.out`
